# ninja_runner
Ninja Runner é um jogo 2D de ação desenvolvido em Python com Pygame, onde o jogador controla um ninja que enfrenta inimigos, desvia de ataques e derrota chefes em fases progressivas. O jogo possui sistema de combate, habilidades especiais, menu interativo, efeitos sonoros, animações e condições de vitória e derrota.
