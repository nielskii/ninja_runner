# 📖 Sobre o Projeto

**Ninja Runner** é um jogo 2D de ação desenvolvido utilizando **Python** e **Pygame** para a disciplina de **Linguagem de Programação Aplicada**.

O jogador controla um ninja que precisa enfrentar inimigos, desviar de ataques, utilizar habilidades especiais e derrotar chefes em fases progressivas.

O projeto possui:
- Sistema de combate
- Boss fight
- Habilidades especiais
- Menu interativo
- Sistema de vida
- Animações
- Áudios e músicas
- Tela de vitória e derrota
- Interface estilizada

---

# 🎮 Gameplay

## Objetivo
Derrotar os inimigos e sobreviver até o final das fases.

# ▶️ Como Executar o Projeto

## 1. Clone o repositório

```bash
git clone https://github.com/nielskii/NinjaRunner.git
```

---

## 2. Entre na pasta do projeto

```bash
cd NinjaRunner
```

---

## 3. Instale as dependências

```bash
pip install pygame
```

---

## 4. Execute o jogo

```bash
python main.py
```

---

# ⚙️ Requisitos

- Python 3.10+
- Pygame

---

# 🖥️ Como Gerar o `.exe`

## Instale o PyInstaller

```bash
pip install pyinstaller
```

---

## Gere o executável

```bash
pyinstaller --onefile --windowed main.py
```

---

# 📁 IMPORTANTE — Assets

Após gerar o `.exe`, copie as pastas:

```text
audio/
assets/

```

para a mesma pasta do executável.

